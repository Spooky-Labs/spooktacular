## Reporting Bugs

Please use the [bug report template](https://github.com/Spooky-Labs/spooktacular/issues/new?template=bug_report.yml) and include:

1. macOS version and Mac model
2. Spooktacular version (`spook --version`)
3. Steps to reproduce
4. Expected vs actual behavior
5. Relevant logs (`log show --predicate 'subsystem == "com.spooktacular"' --last 5m`)

## Security Issues

**Do not open a public issue for security vulnerabilities.** See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
