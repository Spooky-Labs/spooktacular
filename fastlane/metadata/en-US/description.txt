Spooktacular is the fastest way to create, provision, and orchestrate macOS virtual machines on Apple Silicon. Built on Apple's Virtualization.framework for native performance, it serves enterprise CI/CD teams and individual developers with equal ease.

DOUBLE YOUR MAC CAPACITY
Run 2 macOS VMs per Mac — the maximum Apple's EULA allows, enforced at the kernel level. Every Mac mini, Mac Studio, or Mac Pro becomes two workloads.

CREATE VMs IN SECONDS
Spin up macOS virtual machines from IPSW restore images with automatic download, SHA-256 content-addressed caching, and HTTP range resume — no more re-downloading a 15 GB IPSW after a dropped connection. Unattended Setup Assistant automation completes initial configuration hands-free on macOS 15 and macOS 26. APFS copy-on-write clones stamp out identical VMs in ~48 ms without duplicating disk space.

4 PROVISIONING MODES
• SSH — execute user-data scripts with real-time output streaming
• Disk-inject — write files directly into the VM disk image
• VirtIO socket — communicate via the built-in guest agent over three scope-separated channels (read-only / runner / break-glass)
• Shared-folder — drop scripts into a watched folder

GITHUB ACTIONS RUNNERS, FIRST-CLASS
Ephemeral, warm-pool, and persistent runner modes. Registration-token hygiene, drain-before-delete on scale-down, and a 10-check warm-pool scrub validator that verifies the guest is clean before the VM returns to the pool.

KUBERNETES OPERATOR
Deploy MacOSVM and RunnerPool custom resources via the included Helm chart. ValidatingWebhookConfiguration enforces tenant/namespace isolation and the 2-VM-per-host cap at admission. NetworkPolicy, PodDisruptionBudget, ServiceMonitor, and PrometheusRule ship with the chart.

ENTERPRISE CONTROLS
OIDC + SAML federated identity with JWKS pinning. Multi-tenant RBAC with an audit record on every decision. Per-VM network egress policies that generate pf(8) rules. Break-glass tickets with Secure-Enclave-bound keys, single-use per-JTI and per-fleet via a distributed singleton.

AUDIT TRAIL
RFC 6962-aligned Merkle tree with Ed25519 signed tree heads. Append-only file store with fsync on every write. The spook audit verify CLI validates inclusion proofs end-to-end. Optional S3 Object Lock with HEAD-after-PUT verification to catch silently-disabled WORM buckets.

HTTP API
REST control plane with request-ID correlation, a standard error envelope, body-size caps, split read/write rate buckets, and slow-loris defense.

OBSERVABILITY
OpenTelemetry trace export on every API request. Prometheus metrics with bounded cardinality. SLO catalog, Grafana dashboards (controller + fleet), and alert rules shipped as code.

NETWORKING AND SNAPSHOTS
NAT, bridged, or isolated per VM. Disk-level snapshot save, restore, and atomic staging to avoid half-restored state.

HEADLESS AND GUI
A native SwiftUI app with macOS 26 Liquid Glass provides full visibility into every VM. The menu-bar extra animates during transitions. Per-VM LaunchDaemon services keep VMs running unattended on servers. The bundled CLI (spook) ships 16 commands with --json on every command, documented exit codes, and NO_COLOR support.

ACCESSIBILITY
Dynamic VoiceOver labels throughout the GUI. Every control, every state change, every notification is accessible.

822 TESTS
Comprehensive coverage across 224 suites. Clean Architecture enforced by build-time checks: SpookCore is Foundation-only.
