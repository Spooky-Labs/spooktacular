Spooktacular is the fastest way to create, provision, and manage macOS virtual machines on Apple Silicon. Built on Apple's Virtualization.framework for native performance, it serves enterprise CI/CD teams and individual developers with equal ease.

CREATE VMs IN SECONDS
Spin up macOS virtual machines from IPSW restore images with automatic download and verification. Unattended Setup Assistant automation completes initial configuration hands-free on macOS 15 and macOS 26. Instant APFS copy-on-write cloning lets you stamp out identical VMs without duplicating disk space.

4 PROVISIONING MODES
- SSH: Execute user-data scripts with real-time output streaming
- Disk-inject: Write files directly into the VM disk image
- VirtIO socket: Communicate via the built-in guest agent
- Shared-folder: Drop scripts into a watched folder for automatic execution

TEMPLATES FOR COMMON WORKFLOWS
Run --github-runner, --remote-desktop, or --openclaw to generate turnkey provisioning scripts. GitHub Actions runners support ephemeral mode: the VM auto-destroys when the job completes, giving every build a pristine environment.

KUBERNETES OPERATOR
Deploy MacOSVM custom resources via the included Helm chart. The Swift-based controller reconciles desired state, provisions runners, and manages the full VM lifecycle from your cluster.

HTTP API
9 REST endpoints expose every VM operation for integration with your own tooling, dashboards, or orchestration systems.

NETWORKING AND SNAPSHOTS
Choose NAT, bridged, or isolated networking per VM. Save and restore disk-level snapshots to checkpoint work or roll back changes instantly.

HEADLESS AND GUI
A native SwiftUI app with Liquid Glass design (macOS 26+) provides full visibility into every VM. A menu bar extra offers quick access. For servers, per-VM LaunchDaemon services keep VMs running unattended. The bundled CLI (spook) exposes 14 commands with --json output, --field extraction, and NO_COLOR support.

ACCESSIBILITY
Full VoiceOver support throughout the GUI. Every control, every state change, every notification is accessible.

318 TESTS
Comprehensive test coverage across the entire stack.
