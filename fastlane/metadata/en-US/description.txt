Spooktacular is a native macOS app for creating, running, and managing macOS virtual machines on Apple Silicon. Built directly on Apple's Virtualization.framework, it serves individual developers and small teams who need real macOS VMs — not shared runners, not containers.

DOUBLE YOUR MAC CAPACITY
Run 2 macOS VMs per Mac — the maximum Apple's EULA allows, enforced at the kernel level. Every Mac mini, Mac Studio, or Mac Pro becomes two independent workloads.

CREATE VMs IN SECONDS
Spin up macOS virtual machines from IPSW restore images with automatic download, content-addressed caching, and resumable transfers — no re-downloading a multi-gigabyte IPSW after a dropped connection. Unattended Setup Assistant automation gets a VM to a usable desktop hands-free. APFS copy-on-write clones stamp out identical VMs in milliseconds without duplicating disk space.

ZERO-TOUCH GITHUB ACTIONS RUNNERS
One command creates a macOS VM, registers it as an ephemeral GitHub Actions runner using a personal access token stored in the Keychain — never a CLI flag, env var, or file, so it never touches disk or the process list — boots it headless, and polls GitHub until the runner reports online. No separate registration step.

PROVISION YOUR WAY
Run scripts over SSH with real-time output streaming, or inject files directly into the VM disk image before first boot.

REMOTE DESKTOP AND CLIPBOARD
First-boot templates for Screen Sharing (VNC) remote desktops and the OpenClaw AI agent, plus Guest Tools' SPICE-based clipboard sync between host and guest.

SECURITY BUILT IN
mTLS with TLS 1.3 on the control plane, multi-tenant RBAC with deny-by-default policy, Secure-Enclave-backed break-glass access for emergency operations, per-action MFA via Local Authentication, and an append-only local audit log.

HTTP API
A REST control plane covering VM lifecycle, snapshots, IAM, and RBAC, with OpenTelemetry tracing and Prometheus metrics on every request.

NETWORKING AND SNAPSHOTS
NAT, bridged, or isolated networking per VM. Disk-level snapshot save, restore, and atomic staging to avoid half-restored state.

HEADLESS AND GUI
A native SwiftUI app with macOS 26 Liquid Glass gives full visibility into every VM, plus a menu-bar extra. Per-VM LaunchDaemon services keep VMs running unattended on headless servers. The bundled CLI (spook) covers the same surface with --json output on every command, documented exit codes, and NO_COLOR support.

ACCESSIBILITY
Dynamic VoiceOver labels throughout the GUI. Every control, every state change, every notification is accessible.

PERFECT FOR
• GitHub Actions self-hosted runners, zero-touch
• iOS and macOS development and testing across OS versions
• Remote development environments and interactive remote desktops
• Computer-using AI agents inside a real macOS guest

Spooktacular works only on Apple Silicon Macs. MIT licensed, $0 forever — audit every line of source code.
