Spooktacular is the easiest way to create and manage macOS virtual machines on Apple Silicon. Built for enterprise teams and individual developers alike.

CREATE VMs IN SECONDS
Spin up macOS virtual machines from IPSW restore images. Every VM is a clean, isolated macOS environment.

DOUBLE YOUR CI CAPACITY
Run 2 VMs per Mac — instantly double your GitHub Actions, Buildkite, or CircleCI runner capacity on the same hardware. Cut your EC2 Mac costs in half.

ENTERPRISE-READY
Full CLI for automation, full GUI for visibility. Manage headless servers with the built-in LaunchDaemon service.

PROVISIONING
Two provisioning modes today: SSH-based user-data execution with output streaming, and disk-inject (in progress). Two more planned: guest agent and shared-folder watcher.

TEMPLATES
Built-in templates for GitHub Actions runners, remote desktop, and OpenClaw generate setup scripts you apply via the CLI. Create the VM, then start it with --user-data and --provision ssh.

BUILT ON APPLE'S VIRTUALIZATION FRAMEWORK
Native Apple Silicon performance. Metal GPU acceleration for ML workloads. VirtIO networking, shared folders, and audio. Save and restore VM disk snapshots.

KEY FEATURES
- Create VMs from IPSW restore images
- Instant APFS copy-on-write cloning
- SSH provisioning with user-data scripts
- IP resolution via DHCP leases and ARP table
- GitHub Actions runner template (generates setup script)
- Remote desktop template (generates Screen Sharing setup script)
- NAT, bridged, and isolated networking
- Save/restore disk-level VM snapshots
- 2-VM capacity enforcement
- LaunchDaemon service for headless operation
- Menu bar quick access
- Full CLI with styled terminal output
- Accessibility: complete VoiceOver support

PERFECT FOR
- CI/CD runners (GitHub Actions, Buildkite, CircleCI)
- macOS development and testing across versions
- ML training and inference on Apple Silicon
- Remote development environments
