Spooktacular is the easiest way to create and manage macOS virtual machines on Apple Silicon. Built for enterprise teams and individual developers alike.

CREATE VMs IN SECONDS
Spin up macOS virtual machines from IPSW restore images or pre-built OCI images. Every VM is a clean, isolated macOS environment.

DOUBLE YOUR CI CAPACITY
Run 2 VMs per Mac — instantly double your GitHub Actions, Buildkite, or CircleCI runner capacity on the same hardware. Cut your EC2 Mac costs in half.

ENTERPRISE-READY
Manage VMs as Kubernetes resources with our operator. Scale from 2 to 200 runners with kubectl apply. Full CLI for automation, full GUI for visibility.

PROVISIONING THAT JUST WORKS
Four ways to run scripts on first boot: disk injection (zero-touch), SSH, guest agent, or shared folders. Your user-data scripts run automatically — no manual setup.

BUILT ON APPLE'S VIRTUALIZATION FRAMEWORK
Native Apple Silicon performance. Metal GPU acceleration for ML workloads. VirtIO networking, shared folders, and audio. Save and restore VM state.
