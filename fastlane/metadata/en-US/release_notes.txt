v0.1.0 - First Beta

VM Lifecycle
- Create macOS VMs from IPSW restore images with automatic download
- Unattended Setup Assistant automation (macOS 15 and macOS 26)
- Instant APFS copy-on-write cloning with unique MachineIdentifier
- Start, stop, and destroy VMs with PID tracking and graceful shutdown
- Disk-level snapshot save and restore

Provisioning
- 4 modes: SSH, disk-inject, VirtIO socket (guest agent), shared-folder
- SSH user-data execution with real-time output streaming
- IP resolution via DHCP leases and ARP table

Templates
- --github-runner: GitHub Actions runner with ephemeral mode (auto-destroy on stop)
- --remote-desktop: Screen Sharing setup
- --openclaw: OpenClaw environment

Kubernetes Operator
- MacOSVM custom resource definition
- Helm chart for deployment
- Swift-based controller with full reconciliation loop

HTTP API
- 9 REST endpoints for VM lifecycle, provisioning, and status

Networking
- NAT (default), bridged, and isolated networking modes

Service
- Per-VM LaunchDaemon service generation for headless operation
- Shared folder mount persistence across reboots

CLI
- 14 commands with styled ANSI output
- --json for automation, --field for scripting, NO_COLOR support

GUI
- Native SwiftUI app with Liquid Glass design (macOS 26+)
- Menu bar extra for quick access
- Full VoiceOver and accessibility support

Testing
- 318 tests across the full stack
