Privacy Policy

Last updated: April 12, 2026

Summary

Spooktacular does not collect, transmit, or store any personal data. Everything runs locally on your Mac.

Data Collection

Spooktacular collects no data whatsoever. Specifically:

  • No analytics or telemetry
  • No crash reports sent to us
  • No usage tracking
  • No account creation
  • No network requests to our servers

Local Data

Spooktacular stores virtual machine data locally on your Mac at ~/.spooktacular/. This includes:

  • VM disk images and configuration files
  • Cached IPSW restore images
  • Image library metadata
  • An API token for the local control API (never transmitted)

This data never leaves your machine unless you explicitly push it to a container registry via the spook push command.

Network Connections

Spooktacular makes network connections only when you explicitly request them:

  • IPSW downloads — connects to Apple's CDN to download macOS restore images
  • OCI registry — connects to container registries (GHCR, Docker Hub, etc.) when you push or pull images
  • VM networking — VMs with NAT or bridged networking access the network through your Mac's connection

No connection is ever made to Spooktacular's servers.

Third-Party Services

Spooktacular is open source (MIT license). You can audit the source code to verify these claims.

Children's Privacy

Spooktacular is a developer tool and is not directed at children under 13.

Changes

If this policy changes, we will update this page and the "Last updated" date above.

Contact

Questions about this policy? Open an issue on GitHub.
