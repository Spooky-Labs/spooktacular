Spooktacular creates macOS virtual machines using Apple's Virtualization framework. Your use of macOS within virtual machines is governed by Apple's macOS Software License Agreement, which includes these constraints:

  • 2 VM limit per host — Apple Silicon Macs can run a maximum of 2 macOS VMs concurrently. This is enforced by the macOS kernel.
  • Apple hardware only — macOS VMs may only run on Apple-branded hardware.
  • No service bureau use — macOS VMs may not be used for "service bureau, time-sharing, terminal sharing, relay service, or other similar types of services" (i.e., selling VM compute to third parties).
  • Internal use permitted — Running macOS VMs for your own software development, testing, and internal operations is permitted.

Spooktacular enforces the 2-VM kernel limit automatically. Compliance with the remaining EULA terms is your responsibility.
