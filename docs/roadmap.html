v0.1 – v0.4 — Foundation

Shipped

Core VM Lifecycle

  • ✓ Create VMs from Apple IPSW restore images
  • ✓ Instant APFS copy-on-write cloning with MachineIdentifier regeneration
  • ✓ Start / stop / delete VMs with PID tracking
  • ✓ VM bundle format (config.json + platform artifacts + disk image)
  • ✓ IPSW version compatibility checking (fails fast, not after 13 GB download)
  • ✓ 2-VM capacity enforcement via kernel limit detection

Hardware & Networking

  • ✓ CPU, memory, disk, 1–2 Metal GPU displays
  • ✓ Audio (speaker + microphone), clipboard sharing
  • ✓ Shared folders via VirtIO file system with automount
  • ✓ NAT, bridged, isolated, and host-only networking
  • ✓ MAC address configuration, auto-resize display (macOS 14+)

CLI & GUI

  • ✓ 16 CLI commands with styled ANSI output (respects NO_COLOR)
  • ✓ SwiftUI GUI: NavigationSplitView, inspector, create sheet, menu bar extra
  • ✓ Liquid Glass on macOS 26+ with automatic fallback
  • ✓ Full accessibility: VoiceOver labels, WCAG 2.1, reduced motion
  • ✓ Image library browser in sidebar

Quality & Distribution

  • ✓ 235 tests across 33 suites (TDD throughout)
  • ✓ DocC documentation catalog with 11 guides + full API reference
  • ✓ Fastlane CI/CD pipeline (test, build, beta, release, docs, lint)
  • ✓ GitHub Actions CI with GitHub Flow
  • ✓ GitHub Pages site with compare, roadmap, features
