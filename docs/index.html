Why Spooktacular?

2x Your CI Capacity

Run 2 macOS VMs per Mac — double your GitHub Actions runners on EC2 Mac for the same cost.

Enterprise Kubernetes

Manage VMs as K8s resources. kubectl apply creates runners. Scale from 2 to 200.

Zero-Touch Provisioning

User-data scripts run automatically on boot. No SSH, no agent, no manual setup required.

ML on Apple Silicon

Metal GPU acceleration, shared folders for datasets, host-only networking for federated learning.

Remote Desktop

Screen Sharing built in. Connect from anywhere via VNC.

CLI + GUI + API

16 CLI commands, native SwiftUI app, HTTP control API, menu bar extra. Your choice.

Built with Apple's Virtualization.framework. Swift 6. 89 tests. DocC documented.
